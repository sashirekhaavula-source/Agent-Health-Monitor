# Agent Health Monitor

A meta-agent that monitors the health of other AI agents in real time.

## What it does
- Evaluates agent performance using Claude (Anthropic)
- Scores agent health from 0-100
- Logs all traces, costs and latency to Langfuse for observability

## Tools used
- Claude API (Anthropic) — AI evaluation engine
- Langfuse — agent observability and tracing
- Python — core language
